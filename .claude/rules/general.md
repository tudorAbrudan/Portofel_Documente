# Reguli generale

- **Limbă UI:** Toate textele vizibile utilizatorului sunt în română.
- **Stack-ul unic:** React Native + Expo (TypeScript). Nu folosi React Native CLI direct.
- **Local-first:** Niciun backend, nicio autentificare online. SQLite local + fișiere locale. Backup = Drive/iCloud.
- **Expo commands:** `npx expo start --clear` (nu `react-native start`).
- **Plan first:** La orice task cu 3+ pași sau mai multe fișiere: citește, listează fișierele, prezintă planul, implementează.
- **Done = dovedit:** UI → vizibil pe device/emulator. Logică → teste.
